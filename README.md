# 🌱 Projeto FarmTech Solutions – Irrigação Inteligente

## 📌 Descrição

Este projeto tem como objetivo simular um sistema de irrigação automatizado utilizando ESP32, baseado em sensores simulados na plataforma Wokwi.

---

## 🌿 Cultura escolhida

Cana-de-açúcar

A escolha foi feita por ser uma cultura amplamente presente no estado de Sergipe, tornando o projeto mais alinhado com a realidade regional.

---

## ⚙️ Componentes utilizados

- ESP32
- 3 botões (NPK)
- Sensor DHT22 (umidade)
- Sensor LDR (simulando pH)
- Relé (bomba de irrigação)

---

## 🧠 Lógica do sistema

A irrigação é ativada quando:

- Umidade < 65%
- pH dentro da faixa adequada
- Pelo menos 2 nutrientes ativos

---

## 🔄 Funcionamento

O sistema realiza leituras contínuas dos sensores e toma decisões automáticas para ligar ou desligar a irrigação.
